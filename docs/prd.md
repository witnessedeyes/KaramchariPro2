# KaramchariPro网站需求文档
\n## 1. 网站名称
KaramchariPro\n
## 2. 网站描述
KaramchariPro是为印度非正式劳动力打造的经济基础设施平台，定位为薪资支付和劳动力管理系统，而非简单的薪资应用。平台致力于在雇主和工人之间建立数字化信任体系，为印度2亿非正式劳动者提供金融身份和可验证的就业记录。

## 3. 品牌定位
- 这是一个经济基础设施网站，而非薪资应用网站
- 语调：自信、冷静、权威
- 无炒作、无俚语
- 印度优先，关注实际情况
- 核心理念：'我们比银行更了解您的业务'
- 参考风格：Razorpay × UPI × 政府级严肃性

## 5. 网站设计风格

### 5.1 配色方案
- 主色：深海军蓝 #0B1C2D（信任、稳定）
- 辅助色：翡翠绿 #1DB954（金钱、增长）
- 强调色：藏红花金 #F4B400（印度、权威）
- 背景色：柔和灰 #F7F9FC\n- 文字色：炭黑 #1A1A1A

### 5.2 字体设计
- 标题字体：Inter / SF Pro / Manrope（粗体、现代）
- 正文字体：Inter Regular（低端设备高可读性）
- 数字字体：JetBrains Mono（用于财务数据，增强可信度）

### 5.3 整体风格
- 不浮夸、不廉价、不'创业风'\n- 传达必然性和政府级严肃感（但保持私营特质）
- 信任 > 功能 > 增长的优先级
- 视觉元素简洁、结构化、权威
- 适配低端设备，确保在2G网络环境下流畅访问
